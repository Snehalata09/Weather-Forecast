# Weather Forecast App - Android Weather Application

## 📖 About The Project

Weather Forecast App is an Android application developed using Java and Android Studio that provides real-time weather information for cities around the world. Users can search for any city and view current weather conditions, temperature, humidity, wind speed, and other weather-related details through a simple and user-friendly interface.

## 🚀 Features

* Search Weather by City Name
* Real-Time Weather Updates
* Current Temperature Display
* Humidity Information
* Wind Speed Details
* Weather Condition Status
* Sunrise & Sunset Information
* User-Friendly Interface
* Fast Weather Data Retrieval

## 🛠️ Built With

## 📂 Project Modules

### Home Module

Displays weather information and provides access to city search functionality.

### Search Module

Allows users to search weather information for different cities.

### Weather Module

Fetches and displays real-time weather data including temperature, humidity, pressure, and wind speed.

### API Module

Handles communication with the weather service API and processes JSON responses.

## 🎯 Objectives

The primary objective of this project is to provide accurate and real-time weather information to users through a simple Android application, helping them plan their daily activities based on weather conditions.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Weather-Forecast-App.git
```

2. Open the project in Android Studio.

3. Sync Gradle dependencies.

4. Add your Weather API key if required.

5. Run the application on an Android Emulator or Physical Device.
